ShadeQ Editor
========

ShadeQ Level Editor for Lite-C

Controls:
WASD:               Fly
Right Mouse + Move: Look around
Middle Mouse:       Toggle gizmo
1:                  Select Move Gizmo
2:                  Select Rotate Gizmo
3:                  Select Scale Gizmo
Remove:             Delete selection
E:                  Morph selection
Q:                  Clone selection
F6:                 Take screenshot
Left Mouse:         Select
Left Mouse + Shift: Multiselect
